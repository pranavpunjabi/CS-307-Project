Original commands -

Return all users: curl -i http://localhost:5000/server/api/v1.0/users

Return user with a particular ID : curl -i http://localhost:5000/server/api/v1.0/users/n
(n is the ID, eg $ curl -i http://localhost:5000/server/api/v1.0/users/1)

Return user with a particular location : curl -i http://localhost:5000/server/api/v1.0/users/Lafayette

Add new user : curl -i -H "Content-Type: application/json" -X POST -d '{"name":"kumar","location":"west"}' http://localhost:5000/server/api/v1.0/users
